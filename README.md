## üìñ Introduction

**Volunteer AI** is a web-based application designed to support volunteer teachers in under-resourced areas. It moves beyond traditional academic grading by using **Multimodal AI (GLM-4V)** to analyze a student's hidden potential.

By uploading **artwork, audio recordings, and observation notes**, the system generates a comprehensive **Growth Analysis Report**, covering:
* üßò **Potential Profile**: Personality and hidden talents.
* üöÄ **Learning Advice**: Actionable, psychological-based guidance.
* üìä **Competency Radar**: AI-scored metrics on Innovation, Empathy, Drive, etc.
